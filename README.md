Building my own claude code because I don't trust Anthropic no more.

![ex](Example.png)

Clone, put your key and run, no pip install shit.
1. Open the config file: nano ~/.zshrc
2. Add the export line to the bottom: export OPENROUTER_API_KEY="sk-or-..."
3. Save and exit (Ctrl+O, Enter, Ctrl+X).
4. Reload the config: source ~/.zshrc
